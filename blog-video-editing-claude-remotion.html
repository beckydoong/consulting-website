How to Edit Videos in Minutes with Claude Code + Remotion

Seedscale Agency April 13, 2026 6 min read

Stop spending hours manually cutting out silences, retakes, and false starts from your talking-head videos. With Claude Code and Remotion, you can go from raw footage to a polished, captioned short-form video in minutes — with your brand style baked in.

What You'll Need

  • Claude Code — Anthropic's CLI tool
  • Node.js v18+ installed
  • Raw video footage (.mov or .mp4)
  • ~10 minutes for first-time setup, ~5 minutes per video after that

One-Time Setup (10 minutes)

Step 1: Install the Remotion Skill

The Remotion skill gives Claude Code domain-specific knowledge about video editing with Remotion. Install it from GitHub:

# Clone the skill repo
git clone --depth 1 https://github.com/remotion-dev/skills.git /tmp/remotion-skills

# Copy to your Claude skills directory
mkdir -p ~/.claude/skills
cp -r /tmp/remotion-skills/skills/remotion ~/.claude/skills/remotion

# Clean up
rm -rf /tmp/remotion-skills

That's it — Claude Code will automatically detect the skill whenever you work on Remotion projects.

Step 2: Create Your Brand Guide

Inside ~/.claude/skills/remotion/rules/, create a brand-guide.md file that defines your visual style: fonts, colors, layout, title placement, caption styling, and platform safe zones for Instagram and TikTok.

You can ask Claude Code to help you build this — just show it a screenshot of your existing content and say "match this style."

Step 3: Set Up Retake Detection Rules

Create a retake-detection.md rule file that tells Claude how you signal retakes: a keyword like "retake," repeated sentences, false starts, or silence thresholds. Claude Code will follow these rules every time it processes your footage.

Per-Video Workflow (5 minutes)

Step 1: Start a New Conversation

Open Claude Code in your terminal. If you don't have a Remotion project yet, Claude will scaffold one for you.

Step 2: Drop In Your Footage

Tell Claude: "New video — edit this with my brand style" and reference your raw footage file. Claude will:

  1. Copy your video into the project's public/ folder
  2. Analyze the audio — measure loudness levels and detect all silent segments
  3. Transcribe your speech using Whisper (runs locally, no API needed)
  4. Detect cuts — find your "retake" markers, repeated sentences, false starts, and long pauses
  5. Build the composition — stitch together only the good segments
  6. Add branded titles — your fonts, colors, and layout, positioned within platform safe zones
  7. Generate captions — word-by-word with your brand highlight color, timed to your speech
  8. Start the preview — opens Remotion Studio so you can watch the result

Step 3: Review and Adjust

Open http://localhost:3000 to preview in Remotion Studio. Scrub through the timeline to check every cut. Tell Claude to adjust: "move the title up," "that cut at 12s is too early," "change the title text." Changes appear instantly via hot reload.

Step 4: Render

When you're happy, click Render in Remotion Studio or ask Claude to run the render command. Your finished video lands in the out/ folder, ready to upload.

Tips

  • Say "retake" clearly between takes — it's the most reliable cut signal
  • Pause for 1-2 seconds when you mess up before restarting — this helps silence detection catch the boundary
  • Keep your brand guide updated — the more specific it is, the less you'll need to adjust per video
  • Render a single frame first to sanity-check layout before doing a full render
  • Your brand guide, retake rules, and all Remotion skill files persist across conversations — set them up once and they just work
